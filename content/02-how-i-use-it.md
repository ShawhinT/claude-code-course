# How I Use It

Claude Code is versatile. Here are three ways I use it every day.

## Build Products

I use Claude Code to build full applications — web apps, APIs, CLI tools. You describe what you want, and Claude writes the code, creates the files, and sets things up. Products I've built with Claude Code include web apps, browser extensions, and more.

## Build Custom Automations & Tools

Beyond apps, I use Claude Code to build one-off automations — things like YouTube video processing pipelines, markdown-to-PDF converters, and link scrapers. If there's a repetitive task, Claude can usually automate it.

## Ad Hoc Analytics

Got a pile of CSV files? Claude Code can analyze them on the spot. I regularly use it for quick data exploration — no need to set up a Jupyter notebook or write a script from scratch.

> **Note:** Claude Code also handles non-coding tasks (research, writing, and more) — we'll cover that later in the course.
