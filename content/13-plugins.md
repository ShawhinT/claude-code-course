# Plugins

Plugins are collections of skills, MCP servers, subagents, and hooks bundled together. Instead of setting up each piece individually, a plugin installs everything at once.

## How to use plugins

Run the `/plugin` command inside Claude Code:

```bash
/plugin
```

Claude will show you available plugins. Pick one, choose the scope (local, project, or user), and it's installed.

## What's inside a plugin?

A plugin can contain any combination of:

- **Skills** — Reusable instruction sets
- **MCP servers** — External tool connections
- **Subagents** — Specialized agents
- **Hooks** — Automated actions

Think of plugins as "starter kits" that give Claude new capabilities in one step.
