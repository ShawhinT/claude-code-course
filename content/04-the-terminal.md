# The Terminal

The terminal is where you interact with Claude Code. It's a text-based interface on your computer where you type commands and see output.

## How to start Claude Code

Open your terminal, navigate to your project folder, and run:

```bash
claude
```

That's it. Claude starts up and you can begin chatting.

## IDE agnostic

Claude Code runs in the terminal, which means it works alongside **any** editor or IDE — VS Code, Cursor, Vim, Jupyter, or anything else. You're not locked into a specific tool.
