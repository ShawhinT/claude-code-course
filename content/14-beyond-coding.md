# Beyond Coding

Claude Code isn't *just* a coding agent. With the right tools (especially MCP servers), it can handle many non-coding tasks too.

## Research

Claude can search the web, read documents, and compile findings. Point it at URLs, PDFs, or give it a research question and let it dig.

## Writing

Need to draft emails, documentation, blog posts, or reports? Claude can write and edit text — not just code.

## Actions

With MCP servers connected, Claude can take actions in external apps:

- **Google Sheets** — Read and update spreadsheets
- **Figma** — Inspect designs
- **Gmail** — Draft and send emails
- **Slack** — Post messages
- **Notion** — Update pages
- ...and much more

> **Pro Tip:** Customize Claude for non-coding tasks via `/output-style`. This adjusts how Claude formats its responses.
