# Takeaways

Here's what we covered in this crash course.

**Claude Code** is a helpful agent you can access via the terminal (and other surfaces).

## Basic Usage

- Manage Claude's **context window** well to get better performance
- **CLAUDE.md** gives Claude persistent memory across sessions
- **Slash commands** give you shortcuts for managing Claude and its context

## Customizing Claude Code

- **Skills** give Claude specialized instructions and tools for specific tasks
- **MCP servers** give Claude access to external tools and context
- Claude can delegate work to fresh, specialized **subagents**

## Optimizing Claude Code

- **Hooks** allow you to automate commands based on triggers
- Skills, MCP servers, subagents, and hooks can be bundled into **Plugins**

> **When in doubt — just ask Claude!**
