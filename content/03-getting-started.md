# Getting Started

Getting Claude Code running takes two steps.

## Step 1: Get a Claude Subscription

Claude Code requires a Claude subscription. Head to [claude.com/pricing](https://claude.com/pricing) and pick a plan.

## Step 2: Install

**macOS, Linux, or WSL:**

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**

```powershell
irm https://claude.ai/install.ps1 | iex
```

That's it. Once installed, you can start Claude Code by running `claude` in your terminal.
